# Data Availability

The CMS Open Data used in this analysis are publicly available through
the CERN Open Data Portal (https://opendata.cern.ch). The specific
dataset is the CMS 2016 DoubleMuon NanoAOD, accessible at the URL
documented in Reference [1]. All analysis scripts, the frozen protocol
document, and SHA-256 checksums for input data files are available
in the accompanying repository at https://github.com/blumhardt-m/zmet-nid,
tagged at the version corresponding to this submission.
