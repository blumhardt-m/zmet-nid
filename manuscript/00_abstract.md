# Abstract

**Title:** On the interpretability of missing transverse momentum tails with public LHC data: a Z→μ⁺μ⁻ study using CMS Open Data

Missing transverse momentum (MET) is a composite observable whose interpretation depends sensitively on detector response, pileup, and reconstruction choices. Within the CMS Collaboration, MET performance has been extensively characterized using full detector information and dedicated systematic variations. By contrast, external analyses using CMS Open Data are necessarily restricted to reduced-tier representations, such as NanoAOD, which expose MET primarily as a reconstructed end-product.

In this work, we study MET tails in Z→μ⁺μ⁻ events using 2016 CMS Open Data, where genuine MET is expected to be negligible and observed MET is dominated by instrumental effects. Our goal is not to improve MET reconstruction or to replicate collaboration-grade systematics, but to assess the extent to which distinct, physically plausible nuisance explanations can be discriminated using only public-tier observables.

We construct two low-dimensional phenomenological nuisance model families, correlated respectively with jet activity and with pileup proxies, and tune each to reproduce the observed MET distribution. Using preregistered train/test splits and quantitative agreement metrics, we evaluate whether these incompatible explanations can provide comparable agreement with data while passing commonly used global validation checks. We further examine whether stratified diagnostics based on available observables reliably break such degeneracies.

We find that, under the constraints imposed by public-tier representations, multiple nuisance explanations can often reproduce MET tails while remaining difficult to distinguish using standard global validations. These results do not challenge collaboration MET performance studies, but instead quantify an inference boundary relevant to external analyses using open data. The study provides a reproducible framework for assessing identifiability limits of high-level observables in reduced data tiers and informs responsible interpretation of MET in public LHC datasets.
